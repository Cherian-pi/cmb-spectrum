# CMB TT Spectrum from Posterior Parameters (Early Quintessence)

This script plots the CMB TT power spectrum generated by CAMB using a fixed
posterior mean parameter set from a Early Quintessence model. It overlays
the result with Planck 2018 bandpowers.

- Parameters are stored in `params.json`
- Update the FITS file path inside `plot_cmb_from_posterior.py`

This repository **does not** run MCMC or perform parameter inference. 
It only visualizes the spectrum from one chosen parameter point.
